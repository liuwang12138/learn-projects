## 二进制移位操作
```
N << 1			=>		N * 2
N << 1 | 1		=>		N * 2 + 1
```

## 异或运算
不同为1，相同为0
- N ^ 0 == N
- N ^ N == 0
- 满足交换律和结合律

### Question 01

如何不借助其他变量交换a和b？

> a = a ^ b;
> b = a ^ b;
> a = a ^ b;

### Question 02

一个数组中有一种数出现了奇数次，其他数都出现了偶数次，怎么找到并打印这种数？

> 全部异或

### Question 03

怎么把一个整型数，提取出最右侧的1来

> N & (~N + 1)

### Question 04

一个数组中有两种数都出现了奇数次，其他数都出现了偶数次，怎么找到并打印这两种数？

> 不妨设，这两个数字为 a 和 b
> 1. 全部异或起来，由于有两种数都出现了奇数次，所以异或的结果不为0，得到 a ^ b，记为c
> 2. 利用 Question03 找到结果数字最右侧的1
> 3. 所有的数字可以分为两种：一种是该位置为0的，一种是该位置为1的，可以看到，a 和 b 一定分在不同的两侧(如果分在同一侧，则这个位置不可能为1)
> 4. 将一侧的数字全部异或，得到 a（或b）
> 5. 将 c ^ a(或b)，得到b(或a)
